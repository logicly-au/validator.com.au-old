The 2017-18 version of the MHE MDS Data Entry Tool (zip format, 988 kB) is available for download.

If you are downloading this tool for the first time you will also require the "back end" database (zip format, 67 kB). Separating the user interface from the data store in this manner allows us to update the interface without overwriting your data.

These files are distributed in zipped form. Extract them both to the same folder on your computer and open the MHE-NMDS-2017-18-Interface.mdb file with Microsoft Access.

NOTE: If you have entered any data DO NOT extract the back end database (mhe-nmds-2017-2018-empty-data.zip) file into the same directory as your old MHE NMDS database or you will lose your data.

The user guide (pdf format, 584 kB) is available for download.

  • MHE-NMDS Interface (zip format, 570 kB)
